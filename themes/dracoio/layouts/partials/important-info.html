Teiid 4 Ways ...

  • Teiid Wildfly --> Teiid integrated as a module on WildFly Java Application Server and accessing your data services via JDBC/OData/REST APIs.
  • Wildfly Swarm --> Teiid integrated on WildFly Java Application Server to run Teiid as microservices in your applications.
  • Microservices --> Teiid implemented for the Spring Boot framework.
  • Embedded --> A light-weight version of Teiid for use in any Java 7+ JRE application.
